# Changelog

## 1.0.0 — 2026-07-12

- Base Next.js 16, React 19, TypeScript estricto y Tailwind 4.
- Shell responsive en español para plataforma, propietario, recepción y profesor.
- Flujos de alumnos, tutores, cursos, comisiones, inscripción/cargos, pagos, caja, QR, asistencia, solicitudes, alertas, incidentes, liquidaciones, importación e informes.
- Esquema Supabase con migraciones, RLS por tenant/academia/sede, RPC financieras y seed de dos clientes.
- Adaptadores R2, SMTP/outbox y WhatsApp desactivado.
- CSV/XLSX/PDF, health/readiness, cron, logs correlacionados y headers de seguridad.
- Docker multi-stage, Render Blueprint, CI, pruebas unitarias, E2E y pgTAP.
